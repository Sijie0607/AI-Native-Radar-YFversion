# AI-Native 读书雷达 - 首页设计方案

## 1. 首页设计目标与范围边界

### 设计目标
让用户进入页面后，在 10 秒内理解：
1. 这个产品是一个 AI 从业者的读书雷达
2. 雷达上的每个点代表一本书
3. 八大方向代表不同 AI 学习/实践领域
4. 内外圈代表不同阅读难度或进阶阶段
5. 推荐指数和推荐理由帮助用户判断是否值得读

### 范围边界
- **包含内容**：雷达图展示、书籍详情弹窗、基础筛选、导航入口
- **不包含内容**：完整推荐提交页面、用户体系、复杂评分机制、历史版本对比、完整书单列表页（仅保留入口）

---

## 2. 首页信息架构

首页由以下区域组成：
1. **顶部导航区**：品牌标识、导航入口
2. **主视觉区**：雷达图可视化
3. **辅助信息区**：图例说明、筛选器
4. **入口区**：书单列表、推荐书籍入口（仅入口，不展开功能）
5. **详情展示区**：书籍详情（浮层/抽屉形式）

---

```
┌─────────────────────────────────────────────────────────────┐

│                    │                 │                      │
│                    │                 │                      │

│                    │                 │                      │
│                    │                 │                      │

│                                                             │
```

### 布局分区说明
- **首屏**：完整显示导航区 + 雷达图 + 图例

- **辅助区**：图例和筛选器在雷达图右侧和下方

## 4. 核心组件清单

| 组件名称 | 职责边界 | 交互方式 |
|---------|---------|---------|
| 导航栏 | 品牌展示、页面跳转 | 点击跳转 |
| 雷达图组件 | 书籍可视化展示 | 悬停显示摘要、点击进入详情、支持缩放和拖拽平移 |
| 图例说明 | 解释领域分类、难度层级、推荐指数 | 静态展示 |
| 筛选器 | 筛选显示的书籍 | 多选项、实时筛选 |
| 书籍详情卡 | 展示选中书籍的完整信息 | 浮层/抽屉、关闭 |
| 入口按钮 | 引导到其他页面 | 点击跳转 |

---

## 5. 核心用户任务流

### 任务流一：从雷达图探索
用户进入 → 查看雷达图 → 悬停查看书籍摘要 → 点击查看详情 → 关闭详情 → 继续探索

### 任务流二：从筛选探索
用户进入 → 使用筛选器 → 查看筛选后的雷达图 → 悬停查看摘要 → 点击查看详情 → 关闭详情

### 任务流三：从入口跳转
用户进入 → 点击入口按钮 → 跳转到其他页面（提示：功能暂未开放）

---

## 6. 关键交互设计

### 悬停交互（Hover）
- **雷达图点**：悬停放大1.5倍，显示摘要卡片
- **摘要卡片内容**：书名、作者、推荐指数、推荐理由摘要
- **卡片位置**：跟随鼠标，避免超出视口

### 点击交互（Click）
- **雷达图点**：打开书籍详情卡
- **导航链接**：跳转到对应页面
- **筛选选项**：切换选中状态
- **关闭按钮**：关闭详情卡

### 筛选交互
- **领域筛选**：多选，实时更新雷达图
- **推荐指数筛选**：滑块选择，实时更新
- **筛选清空**：一键重置所有筛选

### 缩放与平移
- **缩放**：滚轮缩放雷达图
- **平移**：拖拽平移雷达图

## 7. 页面状态清单

| 状态名称 | 视觉表现 |
|---------|---------|
| 默认状态 | 完整雷达图、所有书籍点可见、无筛选、无详情 |
| 加载状态 | 雷达图显示骨架屏、文字提示「加载中」 |
| Hover状态 | 雷达图点放大、显示摘要卡片 |
| 选中状态 | 详情卡打开、选中的点高亮 |
| 筛选状态 | 只有符合条件的点可见、筛选器显示当前选择 |
| 空结果状态 | 雷达图显示空白、文字提示「无符合条件的书籍」 |
| 错误状态 | 雷达图显示错误图标、文字提示错误信息、重试按钮 |

---

## 8. 状态流转说明

```
默认 → 加载 → 默认
↓
Hover → 默认
↓
Click → 选中 → 默认
↓
筛选 → 筛选状态 → 清空 → 默认
↓
筛选无结果 → 空结果 → 清空 → 默认
↓
出错 → 错误 → 重试 → 默认
```

### 详细状态触发
- **加载状态**：页面初始化触发，数据加载完成后回到默认
- **Hover状态**：鼠标悬停在雷达图点上触发，鼠标离开后回到默认
- **选中状态**：点击雷达图点触发，关闭详情卡后回到默认
- **筛选状态**：点击筛选选项触发，点击清空后回到默认
- **空结果状态**：筛选无结果触发，点击清空后回到默认
- **错误状态**：数据加载失败触发，点击重试后回到加载状态

## 9. 数据与内容字段需求

### 书籍数据字段
| 字段名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| id | String | 是 | 书籍唯一标识 |
| title | String | 是 | 书名 |
| author | String | 是 | 作者 |
| domain | Enum | 是 | 所属领域（8个领域之一） |
| difficulty | Number | 是 | 难度（1-3，1=入门，2=方法，3=进阶） |
| recommendationScore | Number | 是 | 推荐指数（3-5） |
| recommendations | Array | 是 | 推荐理由列表 |
| competenceThemes | Array | 否 | 能力主题标签 |
| coverImage | String | 否 | 封面图片URL |

### 推荐理由字段
| 字段名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| id | String | 是 | 推荐唯一标识 |
| recommender | String | 是 | 推荐人姓名 |
| isAnonymous | Boolean | 是 | 是否匿名 |
| reason | String | 是 | 推荐理由内容 |
| score | Number | 是 | 推荐评分（3-5） |
| recommendedAt | String | 是 | 推荐时间 |
