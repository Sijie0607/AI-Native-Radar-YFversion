# AGENT.md

## 项目定位速览
AI-Native 读书雷达：AI 从业者共建的动态读书雷达，通过主题领域、难度层级、推荐指数和推荐理由，帮助用户快速发现值得读的 AI 相关内容。

## 通用原则
1. 文件缺失或冲突：停止并向我确认，不脑补。
2. 未经确认不写入文件，先在对话中呈现。
3. 结论须可追溯至已确认文件，无支撑需标注为"新增假设"。
4. 只做当前阶段范围内的事，不擅自扩展。

## 分析方法
使用第一性原理思考和拆解问题，推理过程要可见：在给出结论前，先展示拆解过程（至少列出关键的"为什么"链条），而不是只给结论，方便我判断这个结论是真的从根本问题推出来的，还是套用了模板。

## 代码实现
按已确认 spec 实现代码，不做产品或设计决策；不擅自合并 spec 定义的组件边界；不引入 spec 外的第三方库；遇冲突停止并提问，不自行判断。
